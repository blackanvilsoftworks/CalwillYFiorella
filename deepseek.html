Centrar texto en botones Bootstrap

Guía completa para alinear texto vertical y horizontalmente en botones

Por defecto, Bootstrap ya centra el texto en botones tanto vertical como horizontalmente. Pero si necesitas ajustes adicionales, aquí te mostramos cómo hacerlo.

Solución por defecto (funciona sin CSS adicional)

Bootstrap ya incluye estilos para centrado automático:

<button type="button" class="btn btn-primary">Botón primario</button> <button type="button" class="btn btn-success">Botón éxito</button> <button type="button" class="btn btn-danger">Botón peligro</button>

Si necesitas ajustes adicionales

Si por algún motivo necesitas forzar el centrado o ajustar botones personalizados:

1. Usando clases de utilidad de Bootstrap

<!-- Con text-center --> <button type="button" class="btn btn-primary text-center">text-center</button> <!-- Con flexbox --> <button type="button" class="btn btn-success d-flex align-items-center justify-content-center">flexbox</button>

2. Botones de ancho completo

<!-- Con w-100 --> <button type="button" class="btn btn-primary w-100">Botón ancho completo</button> <!-- Con display block --> <button type="button" class="btn btn-success d-block w-100">Botón display block</button>

Soluciones con CSS personalizado

Si necesitas un control más específico, puedes agregar CSS personalizado:

1. Clase personalizada para centrado perfecto

<style> .btn-center-custom { display: flex; align-items: center; justify-content: center; min-height: 48px; } </style> <button type="button" class="btn btn-primary btn-center-custom">Centrado personalizado</button>

2. Centrado para botones con iconos

<style> .btn-icon-center { display: inline-flex; align-items: center; justify-content: center; gap: 8px; } </style> <button type="button" class="btn btn-info btn-icon-center"> <svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" fill="currentColor" class="bi bi-star-fill" viewBox="0 0 16 16"> <path d="M3.612 15.443c-.386.198-.824-.149-.746-.592l.83-4.73L.173 6.765c-.329-.314-.158-.888.283-.95l4.898-.696L7.538.792c.197-.39.73-.39.927 0l2.184 4.327 4.898.696c.441.062.612.636.282.95l-3.522 3.356.83 4.73c.078.443-.36.79-.746.592L8 13.187l-4.389 2.256z"/> </svg> Botón con icono </button>

Problemas comunes y soluciones

Problema: El texto no se centra correctamente

Solución: Verifica que no haya estilos personalizados que estén anulando los estilos de Bootstrap.

Problema: Botones con altura fija

Solución: Usa display: flex junto con align-items: center y justify-content: center.

Problema: Botones con iconos e texto desalineados

Solución: Usa display: inline-flex y gap para espaciado consistente.

Resumen final

En la mayoría de casos, Bootstrap centra automáticamente el texto en botones. Si necesitas ajustes:

  • Usa text-center para centrado horizontal
  • Usa d-flex align-items-center justify-content-center para centrado perfecto
  • Para botones con iconos, usa inline-flex con gap
  • Para problemas específicos, agrega CSS personalizado
