Instalar Vite en tu Proyecto

Guía completa para configurar Vite y acelerar tu desarrollo frontend

¿Qué es Vite?

Vite (del francés "rápido") es una herramienta de construcción frontend que ofrece:

  • Servidor de desarrollo ultrarrápido
  • HMR (Hot Module Replacement) instantáneo
  • Build optimizado para producción
  • Configuración sencilla

Instalación de Vite

1
Abre tu terminal y navega hasta la carpeta de tu proyecto
2
Ejecuta el comando para crear un proyecto Vite
$npm create vite@latest
3
Sigue las instrucciones interactivas:
√ Project name: » mi-proyecto
√ Select a framework: » Vanilla (o elige React, Vue, etc.)
√ Select a variant: » JavaScript (o TypeScript)
4
Navega al directorio del proyecto e instala las dependencias:
$cd mi-proyecto
$npm install

Estructura de archivos después de instalar Vite

mi-proyecto/
├── node_modules/
├── public/
│ └── vite.svg
├── src/
│ ├── assets/
│ ├── App.css
│ ├── App.js
│ ├── index.css
│ └── main.js
├── index.html
├── package.json
├── vite.config.js
└── ...

El archivo index.html ahora es el punto de entrada de tu aplicación, y Vite inyectará automáticamente los scripts necesarios.

Comandos principales de Vite

$npm run dev
# Inicia el servidor de desarrollo en http://localhost:5173
$npm run build
# Construye la versión para producción en la carpeta 'dist'
$npm run preview
# Previsualiza la versión de producción localmente

Configuración adicional para proyectos existentes

Si ya tienes un proyecto y quieres agregar Vite manualmente:

1
Instala Vite como dependencia de desarrollo:
$npm install --save-dev vite
2
Crea un archivo vite.config.js en la raíz:
// vite.config.js import { defineConfig } from 'vite' export default defineConfig({ // Configuración específica de tu proyecto server: { port: 3000, // Puerto personalizado }, build: { outDir: 'dist', // Carpeta de salida para build } })
3
Actualiza tu package.json con los scripts de Vite:
// package.json { "scripts": { "dev": "vite", "build": "vite build", "preview": "vite preview" } }
4
Asegúrate de que tu index.html tenga esta estructura:
<!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Mi Proyecto</title> </head> <body> <div id="app"></div> <script type="module" src="/src/main.js"></script> </body> </html>

Plugins útiles para Vite

Puedes extender la funcionalidad de Vite con plugins:

$npm install --save-dev @vitejs/plugin-react
# Para proyectos React
$npm install --save-dev vite-plugin-pwa
# Para Progressive Web Apps
$npm install --save-dev rollup-plugin-purgecss
# Para eliminar CSS no utilizado

Luego configúralos en tu vite.config.js:

import { defineConfig } from 'vite' import react from '@vitejs/plugin-react' export default defineConfig({ plugins: [react()], })

¡Listo! Vite está instalado

Ahora puedes ejecutar npm run dev para iniciar el servidor de desarrollo y disfrutar de builds ultrarrápidos.

Vite recargará automáticamente tu aplicación cuando detecte cambios en los archivos, lo que acelera significativamente tu flujo de trabajo de desarrollo.
